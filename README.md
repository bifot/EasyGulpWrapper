# Easy Gulp Wrapper

**Делаем все красиво:**

* Клонируем репозиторий

```
git clone https://github.com/bifot/EasyGulpWrapper
```

* Переходим в папку с проектом

```
cd EasyGulpWrapper
```

* Затем устанавливаем все пакеты

```
npm install
```

* Все готово. Запускаем gulp.

```
gulp
```
