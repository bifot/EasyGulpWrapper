# Easy Gulp Boilerplate

* Clone a repository:

```
git clone https://github.com/bifot/easy-gulp-boilerplate
```

* Open folder of project:

```
cd easy-gulp-boilerplate
```

* Install all dependencies:

```
npm i
```

* Run gulp:

```
gulp
```
